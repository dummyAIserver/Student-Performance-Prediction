1. Clone the Repository

2. Create a Virtual Environment
On Windows:
python -m venv venv
venv\Scripts\activate

On macOS / Linux:
python3 -m venv venv
source venv/bin/activate

3. Install Dependencies
All required packages are listed in requirements.txt
pip install -r requirements.txt

4. (Optional) Retrain the Model
If you want to retrain the model, edit the file path in train_model.py (currently uses an absolute path) and run:
python train_model.py

5. Run the Flask Application
python app.py

The server will start in development mode and run at:
http://127.0.0.1:5000/

6. Using the Application

Open the URL in your browser.
Enter attendance, assignment score, and internal marks.
The application will predict the final marks using the pre-trained ML model.
